# Interactive Problems

A type of problems kind of different from the ones you are used to solve on different platforms, and that occur in almost every recent edition of the IOI. They need some prior knowledge about their format to make it easier to solve them. We highly recommmend you to go over the following resources, and solve enough problems to get used to them.
